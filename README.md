# Calarian Connect Showcase Website

This repository contains the source code for the Calarian Connect Showcase Website.

## 🌐 Live Demo
The website is deployed and can be viewed at: [https://mrclumster.github.io/CalarianConnectWeb/](https://mrclumster.github.io/CalarianConnectWeb/)

## 🚀 About the Project
**Calarian Connect** is a modernized digital system designed for Barangay Calarian, Zamboanga City. This showcase website highlights the three main platforms of the system:
1. **Citizen Mobile App** — AI-powered (Voice-to-Form) application for residents.
2. **Staff Web Panel** — Real-time dashboard for barangay administrators.
3. **Face ID Kiosk** — Facial recognition system for fast walk-in identification.

## 🛠️ Tech Stack
This showcase website is built with:
- **Pure HTML/CSS/JS**
- **Tailwind CSS** (via CDN)
- **AOS (Animate On Scroll)** for smooth animations
- **Typed.js** for dynamic text effects
- **FontAwesome** for iconography
