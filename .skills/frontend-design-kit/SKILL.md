# Frontend Design Kit

This skill helps you build "visually complete" and professional-looking web applications.

## Workflow

1.  **Consult Patterns**: Refer to `references/patterns.md` for design fundamentals.
2.  **Evaluate UI**: Check spacing, hierarchy, and color usage in the current code.
3.  **Enhance**: Apply modern CSS techniques (flexbox/grid, shadows, transitions).
4.  **Polish**: Add interactive feedback (hover effects, active states).
