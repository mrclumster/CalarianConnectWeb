# Frontend Design Patterns

## 1. Visual Hierarchy
*   Use font size and weight to draw attention to important elements.
*   Ensure buttons have distinct styles (primary vs secondary).

## 2. Spacing & Rhythm
*   Use a consistent 8px or 4px grid.
*   Provide enough whitespace to let elements "breathe."

## 3. Color & Contrast
*   Ensure text meets WCAG AA accessibility standards.
*   Use a limited palette (1 primary, 1 secondary, 2 neutrals).

## 4. Interaction
*   Provide visual feedback for hover and click states.
*   Use smooth transitions (e.g., `transition: all 0.3s ease`).
