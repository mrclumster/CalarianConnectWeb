# Project Scaffolder

Standardizes project structure to keep it clean and professional.

## Workflow

1.  **Initialize Structure**: Run the scaffolding script.
    ```bash
    node scripts/scaffold.cjs
    ```
2.  **Verify**: Check that `assets/`, `team/`, and `docs/` are created.
3.  **Documentation**: Suggest creating a basic `README.md` if one doesn't exist.
