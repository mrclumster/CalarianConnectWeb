# Backend Optimizer

This skill focuses on making your server-side code and database interactions as fast as possible.

## Workflow

1.  **Analyze Bottlenecks**: Consult `references/tips.md` for common performance issues.
2.  **Audit Code**: Look for N+1 queries, missing indexes, or synchronous blocking calls.
3.  **Propose Fixes**: Suggest database indexing, caching strategies, or architectural changes.
4.  **Validate**: Explain the expected performance gain (e.g., "This changes an O(n²) operation to O(n)").
