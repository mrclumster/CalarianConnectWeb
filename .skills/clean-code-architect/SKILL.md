# Clean Code Architect

This skill ensures your code follows industry standards for readability and maintainability.

## Workflow

1.  **Read Reference**: Consult `references/rules.md` for the core principles.
2.  **Review Code**: Analyze the user's code for DRY violations, long functions, or poor naming.
3.  **Propose Refactor**: Suggest specific changes that align with SOLID principles.
4.  **Explain**: Briefly explain *why* the change makes the code "cleaner."
